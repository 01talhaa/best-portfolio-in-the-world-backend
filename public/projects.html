‚úèÔ∏è 4. Update Project

Endpoint: PUT /api/v1/projects/:id
Headers: 
  Content-Type: application/json
  Authorization: Bearer {access_token}

Request Body: (include only fields you want to update)
{
  "status": "completed",
  "endDate": "2024-03-15",
  "featured": true,
  "technologies": [
    { "name": "React", "category": "Frontend" },
    { "name": "Node.js", "category": "Backend" },
    { "name": "PostgreSQL", "category": "Database" }
  ]
}

Success Response (200):
{
  "success": true,
  "data": {
    "_id": "507f1f77bcf86cd799439011",
    // ... updated project data
    "updatedAt": "2024-03-15T00:00:00.000Z"
  }
}

JavaScript Example:
const updates = {
  status: 'completed',
  featured: true
};

const response = await fetch(`/api/v1/projects/${projectId}`, {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${accessToken}`
  },
  body: JSON.stringify(updates)
});

üì∏ 7. Image Association

Upload images for projects using the upload endpoint:

Endpoint: POST /api/v1/upload/single
Headers: Authorization: Bearer {access_token}
Body: FormData

FormData fields:
- file: (image file)
- entityType: "project"
- entityId: "507f1f77bcf86cd799439011"
- imageType: "thumbnail" or "gallery" (default: "gallery")

JavaScript Example:
const formData = new FormData();
formData.append('file', imageFile);
formData.append('entityType', 'project');
formData.append('entityId', projectId);
formData.append('imageType', 'thumbnail'); // or 'gallery'

const response = await fetch('/api/v1/upload/single', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`
  },
  body: formData
});

// Image will be automatically added to project's thumbnail or images array

üîß Field Reference

Required Fields:
- title: string (unique)
- fullDescription: string
- category: enum ['Website', 'Mobile App', 'Web App', 'E-commerce', 'Residential', 'Commercial', 'Industrial', 'Software', 'Design', 'Consulting', 'Other']

Optional Fields:
- shortDescription: string (max 300 chars)
- featured: boolean (default: false)
- tags: array of strings
- thumbnail: string (URL)
- images: array of strings (URLs)
- videos: array of strings (URLs)
- links: object { demo, github, documentation }
- technologies: array [{ name, category }]
- status: enum ['planning', 'in-progress', 'completed', 'on-hold', 'cancelled'] (default: 'planning')
- client: ObjectId (reference to Client)
- teamMembers: array of ObjectIds (references to TeamMember)
- startDate: Date
- endDate: Date
- budget: object { amount, currency }

Status Values:
- planning: Project is being planned
- in-progress: Development is ongoing
- completed: Project is finished
- on-hold: Temporarily paused
- cancelled: Project was cancelled
